# 🚖 Uber Ride Analytics Dashboard

## 📊 Project Overview
This Power BI dashboard analyzes Uber ride data to uncover booking trends, revenue performance, customer behavior, cancellation patterns, and operational insights.

The goal is to transform raw ride data into meaningful business intelligence using interactive visualizations and KPI tracking.

---

## 🎯 Business Objectives
- Analyze completed and lost bookings
- Track revenue trends across months
- Identify top-performing vehicle types
- Study payment method preferences
- Detect cancellation patterns
- Identify peak demand time slots

---

## 📈 Key KPIs
- 93K Completed Bookings
- 57K Lost Bookings
- 51.85M Revenue
- 2.51M Total Distance
- 24.64 Average Ride Distance

---

## 📊 Dashboard Pages
- Overview
- Vehicle Analysis
- Revenue Analysis
- Rider Analysis
- Location Analysis

---

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
- Interactive Slicers
- KPI Cards

---

## 💡 Key Insights
- Auto generates the highest revenue.
- UPI is the most preferred payment method.
- Evening (6 PM – 9 PM) is peak booking time.
- High lost booking count indicates operational improvement opportunity.
